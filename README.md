# Image cropper

Image cropper im making just for fun, to learn C, and because the usual image editing programs are to annoying to just crop and save the crop as the resolution you want.

The main idea is a simple lightweight image cutting app, where you specify the resolution and just press cut! It will include also some common resolutions (such as profile pictures for specific websites or other similar uses).

Made with [raylib](https://github.com/raysan5/raylib/) and [raygui](https://github.com/raysan5/raygui).


## Building

### Windows

Just run `make all` or `make` and it should work.

### Linux

Make sure to install raylib using your package manager then just run `make all` or `make`.
